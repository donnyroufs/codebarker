# Submit Kata

## Data

### Input

- kataId
- userId
- answerId

### Output

- isCorrect

---

## Pre-condition

- User is authenticated

---

## Primary Course

1. User orders submit kata command
2. System validates input
3. System checks whether the answer is correct
4. System replies with output data

---

## Exception Course

1. Validation
2. Unknown Answer for Kata
3. Unknown Kata

---

## Actor

- User (Detective?)
